## 依赖环境

1. PHP7.0 版本及以上（低版本和7.4版本未做兼容处理！）

## 支持库
- 纯真数据库
- 高德地图
- 淘宝
- 百度搜索
- 腾讯地图
- 太平洋
- 搜狐
- 好搜
- 新浪
- IP Geolocation
- extreme-ip
- ipip
- 网易
- 哔哩哔哩

```text
composer require liguangchun/ip -vvv
```

## 更新

```text
composer update liguangchun/ip -vvv
```

## 删除

```text
composer remove liguangchun/ip -vvv
```
